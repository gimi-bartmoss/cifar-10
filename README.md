# CIFAR-10 Image Classification Project

This is a course project in Python for processing the CIFAR-10 dataset.

## Project Setup

1. **Clone the repository**

    ```bash
    git clone https://github.com/gimi-bartmoss/cifar-10
    cd cifar-10
    ```

1. **Create and activate a virtual environment**

    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

1. **Install dependencies**

    ```bash
    pip install -r requirements.txt
    ```

## How to Run

Ensure the `cifar10.pkl` file is in the same directory as `cifar10.py`, then run:

```bash
python cifar10.py
```
